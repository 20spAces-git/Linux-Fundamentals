# Linux Lab 07 - Bash Basics

## Objective

Learn basic Bash shell concepts and use simple commands to improve efficiency within the Linux command line.

---

## Lab Environment

### Platform

- Linux Terminal

### Commands Used

- echo

---

## Procedure

### Step 1 - Display Text

Command:

echo "Hello Linux"

Purpose:

Displays text to the terminal.

---

### Step 2 - View Command History

Command:

history

Purpose:

Displays previously executed commands.

---

### Step 3 - Clear Terminal

Command:

clear

Purpose:

Clears terminal output.

---

### Step 4 - Access Help Documentation

Command:

man ls

Purpose:

Displays the manual page for a command.

---

### Step 5 - Create a Command Alias

Command:

alias ll='ls -la'

Purpose:

Creates a shortcut for commonly used commands.
